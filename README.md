# Agglomerator

Graphical interface to archive relevant websites to revisit.

</div>

## Usage

The Firefox extension connects to the Tauri instance in your desktop and allows you to add websites while navigating in the browser. Use the `firefox-extension/manifest.json` to load the Firefox plugin in your local instance (`about:debugging`).

To compile the Tauri app for your desktop run the following command in at the root of the project.

```bash
$ npm install
$ npm run tauri build
```
